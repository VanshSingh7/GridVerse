# Gridverse

Real-time board game platform (2-player + VS AI) built with the MERN stack.

## Setup

### Client
cd client
npm install
npm run dev

### Server
cd server
npm install
npm run dev

## Environment Variables (server/.env)
PORT=5000
MONGO_URI=your_mongo_connection_string
JWT_SECRET=your_secret

## Tech Stack
React, Node.js, Express, Socket.io, MongoDB, Mongoose, JWT
